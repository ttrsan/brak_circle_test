ブルアカ風サークルチャット練習サイト

画像配置:
index.html と同じ階層に assets フォルダを置いてください。

構成例:

project/
  index.html
  style.css
  script.js

  assets/
    stamps/
      01/
        01.png
        02.png
      02/
      03/

画像パス:
assets/stamps/01/01.png ～ 20.png
assets/stamps/02/01.png ～ 20.png
assets/stamps/03/01.png ～ 32.png
